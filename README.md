# Thermite Earth Core

A visually rich, single-page educational website exploring the deep connections between thermite reaction physics, Earth's molten core, planetary formation theory, atomic mirror-stream models, aurora color generation, and radon as natural free energy from the planet's interior.

## Live Site

Open `index.html` in any modern browser — no build step required.

## Sections

| Section | Topic |
|---------|-------|
| **Thermite Science** | Nano vs. micro particle size, burn rates, bimodal mixing |
| **Reaction Stages** | Solid–Solid, Solid–Liquid, Liquid–Liquid phases |
| **Stoichiometry** | Fe₂O₃/Al mass ratio, interactive slider, metal-oxide comparison |
| **Earth's Core** | Iron catastrophe, sustained heat sources, dynamo effect |
| **Planetary Nodes** | Sequential solar combustion chambers, condensation sequence |
| **Atomic Theory** | Mirror stream model — charge as direction, not substance |
| **Aurora & Color** | Rayleigh scattering, STEVE, the color-friction model |
| **Radon** | Uranium decay chain, alpha-voltaic energy, health risks |
| **References** | 20 peer-reviewed and government sources |

## Tech Stack

- Vanilla HTML5 / CSS3 / JavaScript (no frameworks or build tools)
- Animated particle canvas background
- IntersectionObserver-driven scroll animations
- Interactive stoichiometry slider
- Animated Earth cross-section diagram
- Responsive layout for mobile and desktop
